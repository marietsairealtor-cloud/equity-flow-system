Exemption only applies to ci_ship_guard.ps1 clean tree check. cloud_migration_parity.json is already robot-owned per 10.8.6A and is regenerated deterministically by sync_truth_registries.mjs. No enforcement rules weakened.

## Risk
Low. Exemption matches existing pattern for handoff_latest.txt and write_path_registry.json which are already exempted for the same reason.
